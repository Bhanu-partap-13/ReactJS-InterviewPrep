## üöÄ Getting Started

### Prerequisites
- Node.js (v14 or higher) - [Download here](https://nodejs.org/)
- npm (comes with Node.js) or yarn

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd ReactJS-ClassWork
   ```

2. **Navigate to the vite project**
   ```bash
   cd "React JS/vite/vite-project"
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**
   - The app will run on `http://localhost:5173`
   - You'll see all the React components and examples
